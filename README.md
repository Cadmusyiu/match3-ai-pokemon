# 🎮 Pokemon Match 3

A 神魔之塔 (Tower of Saviours) / Puzzle & Dragons style Match-3 game featuring Pokemon characters!

## How to Play

### 🎯 Classic Mode
Drag through gems to shift them along your path!
- Gems shift positions as you move through them
- Match 3+ same Pokemon type to score!

### 💡 Hint Mode
AI shows you a good move when stuck!

## Features

- ⚡ **Path Drawing Mechanic** - Like 神魔之塔 / Puzzle & Dragons
- 🔄 **Gem Shifting** - Gems move along your drag path
- 🎮 **Pokemon Characters** - Pikachu, Charmander, Squirtle, and more!
- 💥 **Combo System** - Chain matches for bonus points!
- 📱 **Touch + Mouse** - Works on mobile and desktop!

## Game Rules

1. **Drag** through gems along your desired path
2. Gems **shift positions** as you move (like Tower of Saviours)
3. Match **3 or more** same type Pokemon to score
4. Longer chains = higher combos = more points!

## Tech

- Pure HTML5 + CSS3 + JavaScript (no frameworks!)
- Canvas for path visualization
- Works offline
